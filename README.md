# Website-Design-WHS-Assignment-5
Assignment 5 was a textbook tutorial assignment, meant to teach features to the students in class. The site is for a fictional business called 'Tri and Succeed Sports', or TSS for short.

*Running this File:*
- Clone files as a ZIP archive
- Unzip the archive in a new folder
- Run the "tss_home.html" file to launch the site
